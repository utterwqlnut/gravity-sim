# N-body Real-Time Gravity Simulator
- Implemented in C with the raylib library
- Uses viridis colormap
- Performs Leapfrog integration

## Requirments
- Make sure to install raylib

## How to use
```
gcc main.c -o a -lraylib -g
./a
```
